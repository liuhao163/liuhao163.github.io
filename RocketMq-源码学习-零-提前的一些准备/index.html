зңӢжәҗз Ғзҡ„дёҖдәӣеҮҶеӨҮе·ҘдҪңпјҢд»ҘдёӢйҖ»иҫ‘дјҡеҮәзҺ°еӨҡж¬ЎгҖӮ

RemotingServer

rocketMqйҮҮз”ЁNettyRemotingServerжқҘеӨ„зҗҶеҗ„дёӘз»„д»¶д№Ӣй—ҙзҡ„зҪ‘з»ңйҖҡдҝЎгҖӮ

еӨ„зҗҶж¶ҲжҒҜзҡ„ж–№ејҸ

NettyRemotingServerзҡ„channelRead0()вҖ“>NettyServerHandler.processMessageReceivedвҖ“>NettyServerHandler.processRequestCommandзҡ„processorTableеӨ„зҗҶпјҢе…¶дёӯprocessorTableзҡ„ж–№жі•жҳҜйҖҡиҝҮдёӢйқўзҡ„ж–№ејҸжіЁеҶҢиҝӣеҺ»

  • жіЁеҶҢж¶ҲжҒҜзҡ„пјҡйҖҡиҝҮregisterProcessorж–№жі•пјҢжҜҸз§Қж¶ҲжҒҜж №жҚ®requestcodeпјҢжқҘеҲ¶е®ҡж¶ҲжҒҜеӨ„зҗҶж–№ејҸгҖӮ
    • з»ҙжҠӨдәҶprocessorTableеҜ№иұЎпјҢHashMap<Integer/ request code /, Pair<NettyRequestProcessor, ExecutorService>>
    • жҜҸдёҖз§ҚrequestжҢҮе®ҡдёҖдёӘExecutorServiceе’ҢдёҖдёӘregisterProcessorжқҘеӨ„зҗҶ
  • еӨ„зҗҶж¶ҲжҒҜпјҡйҖҡиҝҮи°ғз”ЁprocessMessageReceivedжқҘеӨ„зҗҶгҖӮ

NettyClientдёүз§ҚеҸ‘йҖҒж¶ҲжҒҜзҡ„ж–№ејҸ

и§ҒдҝЎжҒҜзҡ„еҸ‘йҖҒ

invokeOneway:еҚ•еҗ‘еҸ‘йҖҒж— иҝ”еӣһеҖј

invokeAsync:ејӮжӯҘеҸ‘йҖҒж–№ејҸпјҢinvokeCallbackдёӯеӨ„зҗҶиҝ”еӣһеҖјдҝЎжҒҜ

invokeSync:еҗҢжӯҘеҸ‘йҖҒпјҢзӯүеҫ…иҝ”еӣһеҖј
