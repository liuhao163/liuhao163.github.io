з”ұдәҺеҠЁжҖҒеҲҶжҙҫйңҖиҰҒеңЁclassе…ғж•°жҚ®дёӯеҲҶжһҗжҹҘжүҫж–№жі•пјҢдёәдәҶжҸҗй«ҳжҖ§иғҪпјҢjvmеңЁж–№жі•еҢәз®ҖеҺҶдәҶvtable(иҷҡж–№жі•иЎЁ)пјҢеҪ“и°ғз”ЁжҢҮд»Өinvokeinterfaceж—¶еҖҷдјҡзӣҙжҺҘйҖҡиҝҮvtableзҡ„зҙўеј•жҹҘжүҫйңҖиҰҒзҡ„ж–№жі•зүҲжң¬

vtableвҖ“virtual interface

иҷҡж–№жі•иЎЁпјҢз”ұдәҺеҠЁжҖҒеҲҶжҙҫйқһеёёйў‘з№ҒпјҢjvmдёәдәҶжҸҗй«ҳжҖ§иғҪпјҢеңЁж–№жі•еҢәдёәзұ»еҲӣе»әдәҶиҷҡж–№жі•иЎЁпјҢиҷҡж–№жі•иЎЁдҝқеӯҳзҡ„жҳҜзұ»ж–№жі•зҡ„зңҹе®һең°еқҖпјҢеҰӮжһңйҒҮеҲ°жІЎйҮҚеҶҷзҲ¶зұ»зҡ„ж–№жі•ең°еқҖе°ұжҳҜзҲ¶зұ»зҡ„ең°еқҖгҖӮдёҖиҲ¬еңЁзұ»еҠ иҪҪзҡ„й“ҫжҺҘйҳ¶ж®өпјҢеҪ“зұ»еҲқе§ӢеҢ–0еҖје®ҢжҲҗж–№жі•иЎЁд№ҹеҲқе§ӢеҢ–е®ҢжҲҗ

еҠЁжҖҒзұ»еһӢиҜӯиЁҖж”ҜжҢҒ

еҠЁжҖҒзұ»еһӢиҜӯиЁҖдё»иҰҒжҳҜйҖҡиҝҮjava.lang.invokeеҢ…ж”ҜжҢҒгҖӮд»–е’ҢеҸҚе°„зҡ„еҢәеҲ«жҳҜпјҢеҸ‘е°„ж—¶жЁЎжӢҹзұ»зҡ„еҲӣе»әе’Ңи°ғз”ЁиҝҮзЁӢпјҢиҖҢinvokeжЁЎжӢҹзҡ„жҳҜж“ҚдҪңз Ғ

еҹәдәҺж Ҳзҡ„еӯ—иҠӮз Ғи§ЈйҮҠжү§иЎҢеј•ж“Һ

javaзҡ„жү§иЎҢеј•ж“ҺжҳҜеҹәдәҺж“ҚдҪңж Ҳзҡ„пјҢи§ЈйҮҠжү§иЎҢгҖӮдёҺе…¶зӣёеҜ№зҡ„иҝҳжңүзј–иҜ‘жү§иЎҢеҰӮеӣҫпјҡ

javaзҡ„и§ЈйҮҠеҷЁе’Ңи§ЈйҮҠжү§иЎҢеңЁjvmдёӯпјҢжҠҪиұЎиҜӯжі•ж ‘д№ӢеүҚзҡ„жӯҘйӘӨйғҪзӢ¬з«ӢдәҺjvmиҷҡжӢҹжңәпјҢжүҖд»ҘjavaжҳҜеҚҠзӢ¬з«Ӣи§ЈйҮҠжҖ§иҜӯиЁҖгҖӮ

еҹәдәҺж ҲжҲ–иҖ…еҹәдәҺеҜ„еӯҳеҷЁ

  • еҹәдәҺеҜ„еӯҳеҷЁзҡ„дјҳзјәзӮ№пјҡдјҳзӮ№пјҡзӣҙжҺҘж“ҚдҪңеҜ„еӯҳеҷЁпјҢдёҚзӣҙжҺҘж“ҚдҪңеҶ…еӯҳпјҢCPUжү§иЎҢжҖ§иғҪй«ҳжҖ§иғҪй«ҳгҖӮзјәзӮ№пјҡз”ұдәҺзӣҙжҺҘеңЁеҜ„еӯҳеҷЁж“ҚдҪңдҫқиө–зЎ¬д»¶з§»жӨҚжҖ§е·®
  • еҹәдәҺж Ҳзҡ„дјҳзјәзӮ№пјҡдјҳзӮ№пјҡз§»жӨҚжҖ§еҘҪпјҢйҖҡиҝҮж“ҚдҪңж ҲдёҚзӣҙжҺҘж“ҚдҪңеҜ„еӯҳеҷЁпјҢд»Јз Ғжӣҙдёәзҙ§еҮ‘пјҢз®ҖеҚ•гҖӮзјәзӮ№пјҡж•ҲзҺҮжҜ”еҹәдәҺеҜ„еӯҳеҷЁдҪҺ

и§ЈйҮҠжү§иЎҢиҝҮзЁӢ

е°Ҷж•°жҚ®е…ҲpushеҲ°ж“ҚдҪңж ҲдёӯеңЁpopеҲ°еұҖйғЁеҸҳйҮҸиЎЁпјҢеңЁд»ҺеұҖйғЁеҸҳйҮҸиЎЁloadж•°жҚ®и®Ўз®—з„¶еҗҺpushеҲ°ж“ҚдҪңж ҲдёӯгҖӮ

javaзұ»еҠ иҪҪжңәеҲ¶

Posted on 2019-11-10 | Edited on 2019-11-12 | In java , jvm

JAVAзұ»жҳҜеңЁиҝҗиЎҢж—¶иҝӣиЎҢиҪ¬иҪҪзҡ„пјҢиҝҷз§ҚеҠЁжҖҒжңәеҲ¶иҷҪз„¶йҷҚдҪҺдәҶдәӣи®ёжҖ§иғҪпјҢдҪҶжҳҜдҪҝз”Ёиө·жқҘжӣҙеҠ зҒөжҙ»гҖӮзӣёжҜ”зј–иҜ‘ж—¶еҖҷйңҖиҰҒиҝһжҺҘзҡ„иҜӯиЁҖC++зӯүжқҘиҜҙгҖӮ
зұ»еҠ иҪҪиҝҮзЁӢеҲҶдёәпјҡеҠ иҪҪ-йӘҢиҜҒ-еҮҶеӨҮ-и§Јжһҗ-еҲқе§ӢеҢ–-дҪҝз”Ё-еҚёиҪҪгҖӮе…¶дёӯеҠ иҪҪгҖҒйӘҢиҜҒгҖҒеҲқе§ӢеҢ–зҡ„ејҖе§ӢйЎәеәҸжҳҜеӣәе®ҡпјҢдҪҶжҳҜи§ЈжһҗжңүеҸҜиғҪж”ҫеңЁеҲқе§ӢеҢ–еҗҺйқўгҖӮе№¶дё”д»–д»¬д№ҹжңүеҸҜиғҪжҳҜдәӨеҸүиҝӣиЎҢзҡ„гҖӮ

еҲқе§ӢеҢ–йҳ¶ж®ө

jvmеҜ№еҠ иҪҪжІЎжңүејәеҲ¶зҡ„и§„е®ҡпјҢдҪҶжҳҜеҜ№зұ»зҡ„еҲқе§ӢеҢ–жңүдәҶејәеҲ¶зҡ„и§„е®ҡгҖӮеҚіеҜ№дёҖдёӘзұ»зҡ„дё»еҠЁеј•з”ЁпјҢдё»еҠЁеӣ дёәжҢҮзҡ„жҳҜдёҖдёӢ5дёӯжғ…еҶөгҖӮ

  1. newеҜ№иұЎгҖҒи°ғз”ЁеҜ№иұЎзҡ„staticеұһжҖ§(finalйҷӨеӨ–еӣ дёәfinalзӣҙжҺҘиҝӣе…ҘдәҶиҝҗиЎҢж—¶еёёйҮҸжұ )гҖҒи°ғз”ЁеҜ№иұЎзҡ„staticж–№жі•гҖӮ
  2. еҲқе§ӢеҢ–дёҖдёӘзұ»ж—¶еҖҷеҸ‘зҺ°зҲ¶зұ»жІЎжңүеҲқе§ӢеҢ–иҰҒеҜ№зҲ¶зұ»иҝӣиЎҢеҲқе§ӢеҢ–
  3. mainж–№жі•жүҖд»ҘеңЁзҡ„зұ»
  4. еҸҚе°„ж—¶еҖҷеҰӮжһңеҸ‘зҺ°зұ»жІЎжңүеҲқе§ӢеҢ–

  5. еҪ“дҪҝз”ЁJDK 1.7зҡ„еҠЁжҖҒиҜӯиЁҖж”ҜжҢҒж—¶пјҢеҰӮжһңдёҖдёӘjava.lang.invoke.MethodHandleе®һдҫӢжңҖеҗҺзҡ„и§Јжһҗз»“жһңREF_getStaticгҖҒREF_putStaticгҖҒREF_invokeStaticзҡ„ж–№жі•еҸҘжҹ„пјҢе№¶дё”иҝҷдёӘж–№жі•еҸҘжҹ„жүҖеҜ№еә”зҡ„зұ»жІЎжңүиҝӣиЎҢиҝҮеҲқе§ӢеҢ–пјҢеҲҷйңҖиҰҒе…Ҳи§ҰеҸ‘е…¶еҲқе§ӢеҢ–гҖӮпјҲе®һйҷ…дёҠжҳҜеӣ дёәз¬¬дёҖжқЎзҡ„и§„е®ҡпјү

    жіЁж„ҸпјҡжҺҘеҸЈеҲқе§ӢеҢ–е’Ңзұ»еҫҲзұ»дјјд№ҹйңҖиҰҒеҲқе§ӢеҢ–зҲ¶зұ»жҺҘеҸЈзҡ„classеҜ№иұЎпјҢдёҚиҝҮдёҚйңҖиҰҒеҲқе§ӢеҢ–жүҖжңүзҲ¶зұ»жҺҘеҸЈпјҢеҸӘжңүдҪҝз”ЁзҲ¶зұ»ж—¶еҖҷжүҚдјҡеҲқе§ӢеҢ–пјҲдҪҝз”ЁзҲ¶зұ»жҺҘеҸЈзҡ„еёёйҮҸпјүгҖӮ

еҠ иҪҪиҝҮзЁӢ

еҠ иҪҪ

иҝҷйҮҢеҸӘзҡ„жҳҜзұ»еҠ иҪҪзҡ„вҖңеҠ иҪҪиҝҮзЁӢвҖқгҖӮи®°иҪҪеҲҶдёә3дёӘиҝҮзЁӢ

  1. ж №жҚ®зұ»е…Ёйҷҗе®ҡеҗҚжүҫеҲ°еҜ№еә”еӨ–йғЁдәҢиҝӣеҲ¶ж–Үд»¶
  2. е°ҶдәҢиҝӣеҲ¶ж–Үд»¶еҠ иҪҪеҲ°еҶ…еӯҳдёӯ

  3. еңЁж–№жі•еҢәеҲӣе»әеҜ№еә”зҡ„classзұ»еҒҡеӨ–йғЁи®ҝй—®ж•°жҚ®зҡ„жҺҘеҸЈ

    еҮ з§ҚеӨ–йғЁзҡ„дәҢзә§еҲ¶ж–Үд»¶еҠ иҪҪж–№ејҸzipеҢ…гҖҒзҪ‘з»ңгҖҒиҝҗиЎҢж—¶гҖҒж•°жҚ®еә“зӯүзӯү
    жіЁж„Ҹпјҡж•°з»„зҡ„еҠ иҪҪж–№ејҸпјҢз”ұиҷҡжӢҹжңәзӣҙжҺҘдә§з”ҹгҖӮеҺ»жҺүж•°з»„зҡ„дёҖз»ҙпјҢеҰӮжһңжҳҜеј•з”Ёзұ»еһӢзҡ„иҜқеҚіз»„д»¶зұ»пјҢиҜҘз»„д»¶зұ»зҡ„ClassloaderеҠ иҪҪиҜҘзұ»пјҢе№¶дё”ж•°з»„зұ»дјҡеңЁз»„д»¶зұ»зҡ„Classloaderзҡ„зҡ„зұ»з©әй—ҙиҝӣиЎҢж ҮзӨәгҖӮеҰӮжһңжҳҜintзӯүйқһеј•з”Ёзұ»еһӢпјҢжҲ‘д»¬дјҡи°ғз”Ёеј•еҜјзұ»зҡ„зұ»еҠ иҪҪеҷЁгҖӮи®ҝй—®ж–№ејҸеҸ–еҶідәҺз»„д»¶зұ»зҡ„и®ҝй—®ж–№ејҸгҖӮ

йӘҢиҜҒ

javaиҷҡжӢҹжңәж ЎйӘҢзұ»ж–Үд»¶зҡ„еҗҲжі•жҖ§пјҢйҳІжӯўжҒ¶ж„ҸзҜЎж”№гҖҒзұ»й”ҷиҜҜеҜјиҮҙиҷҡжӢҹжңәжҲ–иҖ…зЁӢеәҸеҙ©жәғгҖӮдё»иҰҒжңүд»ҘдёӢеӣӣдёӘйҳ¶ж®ө

  1. ж–Үд»¶ж јејҸж ЎйӘҢпјҡеҲӨж–ӯclassзҡ„йӯ”ж•°гҖҒзүҲжң¬зӯүпјҢйҖҡиҝҮж ЎйӘҢеҗҺдјҡе°ҶдәҢиҝӣеҲ¶жөҒиҜ»е…ҘеҲ°ж–№жі•еҢәпјҢеҗҺз»ӯзҡ„ж ЎйӘҢдјҡй’ҲеҜ№ж–№жі•еҢәзҡ„дәҢиҝӣеҲ¶жөҒиҝӣиЎҢж ЎйӘҢпјҲжЈҖжҹҘж–Үд»¶пјү
  2. е…ғж•°жҚ®ж ЎйӘҢпјҡеҜ№еӯ—иҠӮз Ғе…ғж•°жҚ®иҝӣиЎҢеҲҶжһҗпјҢжҹҘзңӢжҳҜеҗҰжңүзҲ¶зұ»пјҢзҲ¶зұ»жҳҜеҗҰеҗҲжі•гҖӮпјҲжЈҖжҹҘзұ»пјү
  3. еӯ—иҠӮз Ғж ЎйӘҢпјҡдҝқиҜҒж–№жі•дҪ“еӯ—иҠӮз Ғзҡ„еҗҲжі•жҖ§пјҢжҜ”еҰӮжҳҜеҗҰжңүйқһжі•зҡ„зұ»еһӢиҪ¬жҚўгҖҒиөӢеҖјиғҪгҖӮjdk1.7еҗҺеўһеҠ дәҶStackMapTableз”ЁжқҘдҝқеӯҳж–№жі•дҪ“дёӯеҲқе§Ӣзҡ„зҠ¶жҖҒпјҢж ЎйӘҢж—¶еҖҷзӣҙжҺҘйҖҡиҝҮиҜ»еҸ–StackMapTableеҚіеҸҜпјҲжЈҖжҹҘж–№жі•пјү
  4. з¬ҰеҗҲеј•з”Ёж ЎйӘҢпјҡеӯ—ж®өзҡ„еј•з”ЁжҳҜеҗҰеҗҲжі•пјҢжҳҜеҗҰи®ҝй—®дәҶдёҚеӯҳеңЁзҡ„еӯ—ж®өгҖҒж–№жі•гҖҒд»ҘеҸҠи¶…иҝҮдәҶи®ҝй—®жқғйҷҗзӯү

еҮҶеӨҮ

javaиҷҡжӢҹжңәдёәзұ»зҡ„еҸҳйҮҸеҲҶй…ҚеҶ…еӯҳзҡ„иҝҮзЁӢпјҢиҝҷйҮҢжҢҮзҡ„жҳҜstaticдҝ®йҘ°зҡ„еҸҳйҮҸеӣ дёәе®һдҫӢеҸҳйҮҸдјҡеңЁе®һдҫӢеҲӣе»әж—¶еҖҷиөӢеҖјгҖӮеҜ№дәҺдёҖиҲ¬жғ…еҶөеҸҳйҮҸеҲҶй…Қе®ҢеҶ…еӯҳеҗҺиҰҒиөӢ0еҖјпјҢзңҹе®һзҡ„еҖјдҝқеӯҳеңЁ<clinit>()ж–№жі•дёӯпјҢеңЁеҲқе§ӢеҢ–ж—¶еҖҷжүҚдјҡиөӢеҖјгҖӮfinalдҝ®йҘ°зҡ„еҸҳйҮҸдјҡеңЁиҜҘйҳ¶ж®өиөӢеҖјзңҹе®һеҖј

и§Јжһҗ

javaе°Ҷclassж–Үд»¶дёӯзҡ„з¬ҰеҸ·еј•з”Ёж”№дёәзӣҙжҺҘеј•з”Ёзҡ„иҝҮзЁӢ

еӯ—ж®өзҡ„и§Јжһҗ

  • жҷ®йҖҡеј•з”ЁжҜ”еҰӮеҜ№дәҺеӯ—ж®өNзұ»еһӢCжқҘиҜҙпјҢе°ҶNдәӨз»ҷзұ»зҡ„ClassLoaderеҠ иҪҪCзҡ„зұ»еһӢ
  • ж•°з»„еҠ иҪҪж•°з»„зҡ„ComponentTypeзҡ„зұ»пјҢз„¶еҗҺз”ұиҷҡжӢҹжңәз”ҹжҲҗж•°з»„зҡ„зӣҙжҺҘеә”з”Ё

  • ж ЎйӘҢиҢғеӣҙжқғйҷҗ

    зұ»ж–№жі•зҡ„и§Јжһҗ

  • жЈҖжҹҘзұ»зҡ„class_indexеҰӮжһңжҳҜзұ»ж–№жі•пјҢclass_indexжҳҜжҺҘеҸЈж–№жі•зӣҙжҺҘжҠӣеҮәjava.lang.IncompatibleClassChangeErrorејӮеёёгҖӮ

  • зұ»дёӯжңүж–№жі•зҡ„з®ҖеҚ•еҗҚз§°е’ҢжҸҸиҝ°з¬Ұиҝ”еӣһпјҢеҗҰеҲҷеҺ»зҲ¶зұ»жҹҘжүҫеҰӮжһңжңүиҝ”еӣһпјҢеҗҰеҲҷеҺ»жҺҘеҸЈжҹҘжүҫеҰӮжһңжҺҘеҸЈжңүжҠӣејӮеёёjava.lang.AccessMethodError,еҰӮжһңжІЎжңүжҠӣеҮәjava.lang.NoSuchMethodError

  • жЈҖжҹҘи®ҝй—®жқғйҷҗ

    жҺҘеҸЈж–№жі•и§ЈжһҗпјҡжӯҘйӘӨеҗҢдёҠеҸӘжҳҜдёҚз”ЁжЈҖжҹҘжқғйҷҗ

еҲқе§ӢеҢ–

еңЁиҜҘйҳ¶ж®өејҖе§Ӣжү§иЎҢjavaд»Јз ҒпјҲеӯ—иҠӮз ҒпјүпјҢз”ҹжҲҗж–№жі•<clinit>()гҖӮ

  • clinitж–№жі•жҳҜз”ұйқҷжҖҒеӯ—ж®өе’ҢйқҷжҖҒд»Јз Ғеқ—е…ұеҗҢеҗҲе№¶з”ҹжҲҗпјҢдё”йЎәеәҸиҮӘдёҠеҲ°дёӢ
  • еҰӮжһңжңүзҲ¶зұ»пјҢдё”зҲ¶зұ»жңүжңүйқҷжҖҒзұ»е’ҢйқҷжҖҒеӯ—ж®өпјҢзҲ¶зұ»зҡ„clinitж–№жі•дјҡе…Ҳи°ғз”Ё
  • жҺҘеҸЈеӣ дёәд№ҹеҸҜиғҪжңүйқҷжҖҒйҳ¶ж®өпјҢжҺҘеҸЈд№ҹеҸҜиғҪз”ҹжҲҗclinitпјҢеҰӮжһңжңүзҲ¶зұ»жҺҘеҸЈпјҢеҸӘжңүз”ЁеҲ°зҲ¶зұ»жҺҘеҸЈзҡ„еҸҳйҮҸжүҚдјҡз”ҹжҲҗзҲ¶зұ»жҺҘеҸЈзҡ„clinitж–№жі•

жіЁж„ҸclinitеҮәдәҺзәҝзЁӢе®үе…ЁиҖғиҷ‘дјҡеҠ й”Ғиҝҷе°ұжҳҜдёәд»Җд№Ҳstaticд»Јз Ғеқ—еҸӘжү§иЎҢдёҖж¬ЎгҖӮеҰӮжһңstaticйҮҢйҳ»еЎһдјҡйҳ»еЎһзұ»зҡ„еҲӣе»ә

зұ»еҠ иҪҪеҷЁ

е”ҜдёҖжҖ§пјҡClassLoader+зұ»=е”ҜдёҖпјҢжҜҸдёӘClassLoaderйғҪжңүиҮӘе·ұзҡ„зұ»е‘ҪеҗҚз©әй—ҙпјҢиҝҷдёӘдјҡеҪұе“ҚеҲ°instanceOfзӯүз»“жһңгҖӮ

javaзҡ„classloaderеҲҶдёә3з§Қ

  1. еҗҜеҠЁзұ»еҠ иҪҪеҷЁпјҡBootstrap ClassLoaderпјҢз”ұC++зј–еҶҷпјҢjavaж— жі•зӣҙжҺҘи°ғз”ЁпјҢеҰӮжһңе°Ҷзұ»е§”жүҳз»ҷиҜҘеҠ иҪҪеҷЁи°ғз”Ёж”№classloaderеӣһжҠҘејӮеёёгҖӮз”ЁдәҺеҠ иҪҪJAVA_HOME/libд»ҘеҸҠ -XbootclasspathеҜ№еә”ең°еқҖзҡ„зұ»е№¶дё”иҰҒз¬ҰеҗҲе‘ҪеҗҚи§„еҲҷпјҢжҜ”еҰӮrt.jar
  2. жү©еұ•зұ»еҠ иҪҪеҷЁпјҡExtension ClassLoaderпјҢз”ұjavaзј–еҶҷпјҢjavaеҸҜзӣҙжҺҘи°ғз”ЁгҖӮJAVA_HOME/ext/libжҲ–иҖ…java.ext.dirsжҢҮе®ҡзҡ„зұ»
  3. еә”з”Ёзұ»еҠ иҪҪеҷЁпјҡApplication ClassLoaderпјҢз”ұjavaзј–еҶҷпјҢjavaеҸҜзӣҙжҺҘи°ғз”ЁгҖӮclasspathдёӯзҡ„зұ»пјҢеҰӮжһңдёҚжҢҮе®ҡиҮӘе·ұзҡ„classloaderд»–жҳҜй»ҳи®Өзҡ„classloader

зұ»еҠ иҪҪзҡ„еҸҢдәІе§”жҙҫжЁЎеһӢпјҡеҚізұ»еҠ иҪҪеҷЁдјҡдјҳе…Ҳи°ғз”ЁзҲ¶зұ»зҡ„еҠ иҪҪеҷЁпјҢзҲ¶зұ»еҠ иҪҪеҷЁдјҡи°ғз”ЁзӣҙеҲ°еҗҜеҠЁзұ»еҠ иҪҪеҷЁпјҢд№ӢеҗҺеңЁи°ғз”ЁиҮӘе·ұзҡ„зұ»еҠ иҪҪеҷЁгҖӮеҘҪеӨ„еңЁдәҺеҜ№дәҺдёҖдәӣзі»з»ҹзҡ„зұ»е…ЁеұҖеҸӘжңүдёҖдёӘе”ҜдёҖзҡ„Classзұ»пјҲеҸ—е”ҜдёҖжҖ§зәҰжқҹпјүгҖӮйЎәеәҸжҳҜпјҡеҗҜеҠЁзұ»еҠ иҪҪеҷЁвҖ“>жү©еұ•зұ»еҠ иҪҪеҷЁвҖ“>еә”з”Ёзұ»еҠ иҪҪеҷЁвҖ“>иҮӘе®ҡд№үзұ»еҠ иҪҪеҷЁгҖӮ
