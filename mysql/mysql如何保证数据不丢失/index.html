MysqlжҳҜеҰӮдҪ•дҝқиҜҒж•°жҚ®дёҚдёўеӨұзҡ„

Posted on 2019-02-02 | Edited on 2019-03-01 | In Mysql , MysqlеҹәзЎҖ

MysqlжҳҜеҰӮдҪ•дҝқиҜҒж•°жҚ®дёҚдёўеӨұзҡ„

binlog

  1. дәӢеҠЎжү§иЎҢиҝҮзЁӢдёӯbinlogеҶҷе…ҘеҲ°binlog cache,дәӢеҠЎжҸҗдәӨж—¶еҶҷе…ҘеҲ°binlogж–Үд»¶дёӯгҖӮ
  2. дәӢеҠЎзҡ„еҺҹеӯҗжҖ§еҶіе®ҡиҝҷж— и®әдәӢеҠЎжңүеӨҡеӨ§пјҢbinlogcaheйғҪиҰҒдёҖж¬ЎжҖ§е®Ңж•ҙзҡ„еҶҷе…ҘеҲ°binlogж–Үд»¶дёӯпјҢеҶҷе…Ҙж–№ејҸеҰӮдёӢпјҡ
    1. зі»з»ҹдёәжҜҸдёӘbinlog_cacheејҖиҫҹдәҶдёҖзүҮеҶ…еӯҳпјҲжҜҸдёӘзәҝзЁӢйғҪжңүдёҖдёӘпјүпјҢеҸӮж•°жҳҜblong_cache_size,и¶…иҝҮиҝҷдёӘйҳҲеҖјзҡ„дјҡдҝқеӯҳеңЁдёҙж—¶ж–Үд»¶дёӯ
    2. дәӢеҠЎжҸҗдәӨж—¶еҖҷпјҢеҮәдәҺжҖ§иғҪзҡ„иҖғиҷ‘еӣһе°Ҷbionlog_cacheйғҪwriteеҲ°ж–Үд»¶зі»з»ҹзҡ„page_cacheдёӯпјҢеңЁйҖҡиҝҮfsyncеҲ·ж–°еҲ°зЈҒзӣҳж–Үд»¶дёӯгҖӮеҰӮдёӢеӣҫжүҖзӨәжҳҜbinlogзҡ„еҗҢжӯҘжңәеҲ¶гҖӮ

еҰӮеӣҫпјҡ

  • еҸӘжңүfsnycеҲ·зӣҳиҝҷдёҚж“ҚдҪңжүҚдјҡеҚ з”ЁIOPSгҖӮ
  • writeе’Ңfsyncзҡ„ж—¶жңәжҳҜз”Ёbinglog_syncеҸӮж•°жқҘжҺ§еҲ¶зҡ„
    • =0 жҜҸж¬ЎдәӢеҠЎжҸҗдәӨж—¶еҖҷйғҪеҸӘwriteдёҚsync
    • =1 жҜҸж¬ЎдәӢеҠЎжҸҗдәӨйғҪsync
    • =NпјҲN>1пјү жҜҸж¬ЎдәӢеҠЎжҸҗдәӨйғҪwriteпјҢз§Ҝж”’еҲ°Nж—¶еҖҷжүҚдјҡfsync

еңЁIOеҮәзҺ°з“¶йўҲж—¶еҖҷжҲ‘д»¬еҸҜд»Ҙи®ҫзҪ®дёҖдёӘиҫғеӨ§зҡ„еҖјжқҘжҸҗеҚҮжҖ§иғҪпјҲдёӘдәәдёҚжҳҜеҫҲжҺЁиҚҗеҒҡпјҢеҰӮжһңеҮәзҺ°иҝҷз§Қжғ…еҶөе»әи®®д»ҺдёҡеҠЎиҖғиҷ‘дјҳеҢ–ж•°жҚ®еә“пјү

redologзҡ„еҶҷе…ҘжңәеҲ¶

redologд»ҺеҶҷе…ҘеҲ°жңҖз»ҲеҶҷе…ҘеҲ°зЈҒзӣҳдёӯдјҡз»ҸеҺҶеҰӮдёӢзҡ„йҳ¶ж®өпјҡеҶҷе…Ҙredolog-bufferдёӯвҖ“>еңЁдәӢеҠЎжҸҗдәӨгҖҒжҲ–иҖ…дёҖе®ҡж—¶жңәпјҲи§ҒдёӢйқўпјүдёӢпјҡredolog-bufferеҶҷе…ҘеҲ°ж–Үд»¶зі»з»ҹзҡ„pagecache(write)вҖ“>ж–Үд»¶зі»з»ҹpagecacheеҶҷе…ҘеҲ°зЈҒзӣҳдёӯ(fsnyc)пјҢиҝҷж ·зҡ„ж“ҚдҪңзҡ„ж„Ҹд№үжҳҜдёәдәҶжҸҗй«ҳmysqlзҡ„еҗһеҗҗзҡ„пјҢе…·дҪ“зҡ„жңәеҲ¶и§ҒдёӢйқўпјҡ

  1. дёҖдёӘдәӢеҠЎдјҡдә§з”ҹеҫҲеӨҡжқЎredologеҰӮжһңжҜҸж¬ЎзӣҙжҺҘжҢҒд№…еҢ–зЈҒзӣҳдјҡж¶ҲиҖ—еӨ§йҮҸзҡ„зЈҒзӣҳIOпјҢжүҖд»Ҙredologдјҡе…ҲеҶҷе…Ҙredolog-bufferдёӯпјҢд№ӢеҗҺеңЁwriteж–Үд»¶зі»з»ҹзҡ„еҲ°pagecacheдёӯпјҢиҝҷдҝ©жӯҘжҳҜеҶ…еӯҳж“ҚдҪңеҫҲеҝ«гҖӮ
  2. жҳҜеҗҰдјҡеҪұе“Қж•°жҚ®зҡ„жҢҒд№…еҢ–пјҢжҜ”еҰӮmysqlеңЁдәӢеҠЎиҝӣиЎҢдёӯcrashдәҶпјҢиҝҷж—¶еҖҷredolog-bufferдёӯзҡ„ж•°жҚ®дёўеӨұжҖҺд№ҲеҠһпјҹзӯ”жЎҲжҳҜз”ұдәҺдәӢзү©жІЎжңүжҸҗдәӨпјҢжүҖд»ҘдәӢзү©дјҡиҝӣиЎҢеӣһж»ҡгҖӮ
  3. redolog-bufferжҢҒд№…еҢ–зҡ„жқЎд»¶е’ҢжңәеҲ¶пјҡ
    1. еҸ—еҸӮж•°innodb_flush_log_at_tx_commitзҡ„жҺ§еҲ¶пјҡ
      1. =0пјҡжҜҸж¬ЎжҸҗдәӨйғҪеҸӘеҒңз•ҷеңЁredolog-bufferдёӯпјӣ
      2. =1пјҡжҜҸж¬ЎжҸҗдәӨйғҪдјҡжҢҒд№…еҢ–еҲ°зЈҒзӣҳдёӯпјӣ
      3. =2пјҡжҜҸж¬ЎжҸҗдәӨйғҪеҸӘдјҡеҶҷе…ҘеҲ°ж–Үд»¶зҡ„pagecacheдёӯпјӣ
    2. innodbеҗҺеҸ°дјҡжңүдёҖдёӘзәҝзЁӢжҜҸз§’дёҖж¬ЎпјҢдјҡжҠҠredolog-bufferдёӯзҡ„ж—Ҙеҝ—пјҢи°ғз”ЁwriteеҶҷеҲ°pagecaheдёӯпјҢеңЁfsyncеҲ°зЈҒзӣҳдёӯпјӣ
    3. е…¶д»–еңәжҷҜи§ҰеҸ‘redologзҡ„жҢҒд№…еҢ–
      1. еҪ“redolog-bufferеҚ еҲ°дәҶinnodb_log_buffer_sizeзҡ„дёҖеҚҠж—¶еҖҷпјҢдјҡи°ғз”Ёwriteе°Ҷbufferдёӯзҡ„logеҶҷе…ҘеҲ°ж–Үд»¶зі»з»ҹзҡ„pagecacheдёӯ
      2. еҸҰдёҖз§ҚжҳҜе№¶иЎҢдәӢзү©жҸҗдәӨж—¶еҖҷпјҢеҰӮжһңinnodb_flush_log_at_tx_commitи®ҫзҪ®дёә1ж—¶еҖҷдјҡпјҢеҚіж—¶еҪ“еүҚдәӢзү©жІЎжңүcommitпјҢд№ҹдјҡе°ҶredologеҶҷе…ҘеҲ°ж–Үд»¶зі»з»ҹдёӯгҖӮ

еҸҢ1и®ҫзҪ®

еҚіпјҡbinlogsnyc=1пјҢinnodb_flush_log_at_tx_commitд№ҹи®ҫзҪ®дёә1гҖӮ
з”ұдәҺinnodbзҡ„дәӢзү©жҸҗдәӨredologе’ҢbinlogжҳҜ2PCгҖӮ
жүҖд»ҘеҪ“redologеңЁprepareж—¶еҖҷпјҢдёәдәҶж•…йҡңжҒўеӨҚдёҖе®ҡдјҡжҢҒд№…еҢ–дёҖж¬ЎпјҢжүҖд»Ҙиҝҷж—¶еҖҷйңҖиҰҒfsyncеҲ°зЈҒзӣҳдёӯгҖӮ
binlogеңЁиҝӣиЎҢfsyncеҲ°зЈҒзӣҳдёӯ
иҝҷж—¶еҖҷдәӢзү©еңЁredologеңЁcommitж—¶еҖҷпјҢmysqlз”ұдәҺжңүеҘ”жәғжҒўеӨҚжңәеҲ¶е’ҢеҗҺеҸ°зәҝзЁӢжҜҸз§’иҪ®и®ӯдёҖж¬ЎеҲ·зӣҳдјҡи®ӨдёәredologжІЎжңүеҝ…иҰҒеңЁfsyncдёҖж¬ЎеҲ°зЈҒзӣҳдәҶпјҢеҸӘдјҡеҶҷе…ҘеҲ°ж–Үд»¶зҡ„pagecacheдёӯгҖӮ

gourp commitжңәеҲ¶

дёҖдёӘдәӢзү©зҡ„жҸҗдәӨз”ұдәҺredologе’ҢbinglogйғҪиҰҒжҢҒд№…еҢ–пјҢзЈҒзӣҳIOиҝҳжҳҜеҫҲеӨ§MysqlжҳҜеҰӮдҪ•дјҳеҢ–иҝҷйғЁеҲҶе‘ўгҖӮиҝҷйҮҢmysqlйҮҮз”ЁдәҶз»„жҸҗдәӨпјҲgroup commitпјүзҡ„ж–№ејҸгҖӮ
еҰӮеӣҫжүҖзӨәпјҡ

  • йҰ–е…Ҳд»Ӣз»ҚдёӢLSN(log sequence number)пјҢдёҖдёӘеҚ•и°ғйҖ’еўһзҡ„еәҸеҸ·пјҢз”ЁжқҘеҜ№еә”redologзҡ„дёҖдёӘдёӘеҶҷе…ҘзӮ№гҖӮеҪ“з„¶д№ҹдјҡеҶҷе…ҘеҲ°ж•°жҚ®йЎөдёӯпјҢз”ЁдәҺflushи„Ҹзӣҳж—¶еҖҷйҒҝе…ҚйҮҚеӨҚжү§иЎҢпјҲдёҚеңЁи®Ёи®әиҢғеӣҙеҶ…пјү
  • еҰӮеӣҫпјҢеҪ“3дёӘдәӢеҠЎеҗҢж—¶йғҪеҶҷе®Ңredolog-buffer е№¶дё”еӨ„дәҺprepareйҳ¶ж®өж—¶еҖҷпјҢиҝҷж—¶еҖҷе°ұжһ„жҲҗдәҶдёҖдёӘgourp
    1. з¬¬дёҖдёӘе…ҲеҲ°иҫҫзҡ„trx1пјҢжҲҗдёәз»„йҮҢзҡ„leaderпјҢLSN=50;
    2. зӯүtrx1ејҖе§ӢеҶҷзӣҳж—¶еҖҷпјҢз»„йҮҢе·Із»Ҹжңүе…¶д»–дҝ©дёӘдәӢзү©пјҢиҝҷж—¶еҖҷLSN=160;
    3. trx1ејҖе§ӢеҶҷзӣҳпјҢжүҖжңүlsn<160зҡ„ж—Ҙеҝ—йғҪдјҡиў«еҶҷе…ҘеҲ°зЈҒзӣҳдёӯпјӣ
    4. trx3,trx4е°ұеҸҜд»ҘзӣҙжҺҘиҝ”еӣһдәҶпјӣ

жүҖд»ҘеңЁе№¶еҸ‘дәӢзү©дёӯпјҢеҪ“еҶҷе®ҢredologпјҢи¶Ҡжҷҡи°ғз”ЁfsyncпјҢеёҰзҡ„logи¶ҠеӨҡжҖ§иғҪд№ҹе°ұи¶ҠеҘҪпјӣmysqlеңЁиҝҷж–№жі•йҮҮз”Ёзҡ„жҳҜжӢ–ж—¶й—ҙзҡ„зӯ–з•ҘпјҢеҚі:еңЁеҸҢ1й…ҚзҪ®дёӢ
redolog-prepare(write)вҖ“>binglog(write)вҖ“>redolog-prepare(fsync)вҖ“>binlog(fsnyc)вҖ“>redolog commit(write)

еҰӮеӣҫпјҡ

  • еҰӮдёҠеӣҫжүҖзӨәпјҢз”ұдәҺredologеңЁwriteе’ҢfsnycдёӯжңүдёҖдёӘbinlog-writeзҡ„иҝҮзЁӢпјҢжүҖд»ҘеңЁжҢҒд№…еҢ–зЈҒзӣҳж—¶еҖҷдҪ еҸҜд»ҘеёҰдёҠжӣҙеӨҡзҡ„logпјӣ
  • еҸҰеӨ–пјҡbinlogд№ҹеҸҜд»ҘйҮҮз”Ёз»„жҸҗдәӨпјҢеҸӘдёҚиҝҮз”ұдәҺиҝҷдҝ©дёӘйҳ¶ж®өй—ҙйҡ”зҹӯеҸҜиғҪжІЎжңүredologйӮЈд№ҲжҳҺжҳҫ
    • binlogзҡ„gruop commitзҡ„еҸӮж•°еҰӮдёӢпјҡ
      • binlog_group_commit_sync_delay:иЎЁзӨәе»¶иҝҹеӨҡе°‘еҫ®еҰҷеҗҺе°ұдјҡи°ғз”Ёfsnyc
      • binlog_gourp_commit_sync_no_delay_count:иЎЁзӨәзҙҜи®ЎеӨҡе°‘ж¬ЎеҗҺжүҚи°ғз”Ёfsync
      • дҝ©дёӘеҸӮж•°жҳҜorзҡ„е…ізі»пјҢдёҚиҝҮеҰӮжһңbinlog_group_commit_sync_delayи®ҫзҪ®дёә0пјҢbinlog_gourp_commit_sync_no_delay_countе°ұж— ж•ҲдәҶ

з»јдёҠжүҖиҝ°пјҡ

  • mysqlзҡ„WALжңәеҲ¶жҳҜз”ұдәҺredologе’ҢbinlogйғҪжҳҜйЎәеәҸеҶҷпјҢдҝқиҜҒдәҶй«ҳеҗһеҗҗпјӣ
  • еҗҢж—¶йҮҮз”ЁдәҶз»„жҸҗдәӨзҡ„ж–№ејҸпјҢжқҘеҮҸе°‘дәҶIOPS;
