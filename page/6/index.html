зі»з»ҹдјҳеҢ–-CPU-еҰӮдҪ•жҺ’жҹҘCPUдҪҝз”ЁзҺҮеҲ°100%

жҲ‘д»¬еҰӮдҪ•еҺ»жҹҘзңӢCPUзҡ„дҪҝз”ЁзҺҮе‘ўпјҹCPUдҪҝз”ЁзҺҮдёӯзҡ„еҮ дёӘйҮҚиҰҒжҢҮж ҮеҲҶеҲ«д»ЈиЎЁд»Җд№Ҳпјҹ

CPUдҪҝз”ЁзҺҮзӣёе…ізҡ„жҰӮеҝөд»Ӣз»Қ

жҲ‘д»¬еӨҡж ёзҡ„CPUдёәдәҶиғҪвҖңе№¶иЎҢвҖқиҝҗиЎҢжҲ‘д»¬зҡ„зЁӢеәҸпјҢжҲ‘д»¬йңҖиҰҒдёәжҜҸдёӘиҝӣзЁӢеҲҶй…Қжү§иЎҢж—¶й—ҙпјҢжү§иЎҢж—¶й—ҙеҲ°дәҶжҲ‘д»¬дјҡиҝӣиЎҢдёҖж¬Ўдёӯж–ӯпјҢдёӯж–ӯзҡ„йў‘зҺҮжҲ‘д»¬е®ҡд№үдёәhzпјҢе…¶дёӯеҶ…ж ёжҖҒзҡ„дёӯж–ӯжҲ‘д»¬еҸҜд»Ҙи®ҫзҪ®пјҢдёҖиҲ¬жҳҜ100пјҢ250пјҢ1000hzпјҢиЎЁзӨәз§’дёӯж–ӯеӨҡе°‘ж¬ЎпјҢжҲ‘д»¬зҡ„з”ЁжҲ·жҖҒжҳҜжІЎжі•зӣҙжҺҘжҺ§еҲ¶hzзҡ„гҖӮз”ЁжҲ·жҖҒеҲҷи§„е®ҡдёә100HZпјҢд№ҹе°ұжҳҜ10msгҖӮжҲ‘д»¬жҹҘзңӢCPUзҡ„жҢҮж Үзҡ„еҚ•дҪҚе°ұжҳҜ10msгҖӮ

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮжҹҘзңӢ/proc/cpu/statжқҘзңӢcpuзҡ„жҢҮж ҮгҖӮ

е…¶дёӯеҮ дёӘйҮҚиҰҒжҢҮж Ү

  • userпјҲйҖҡеёёзј©еҶҷдёә usпјүпјҢд»ЈиЎЁз”ЁжҲ·жҖҒ CPU ж—¶й—ҙгҖӮжіЁж„ҸпјҢе®ғдёҚеҢ…жӢ¬дёӢйқўзҡ„ nice ж—¶й—ҙпјҢдҪҶеҢ…жӢ¬дәҶ guest ж—¶й—ҙгҖӮ
  • niceпјҲйҖҡеёёзј©еҶҷдёә niпјүпјҢд»ЈиЎЁдҪҺдјҳе…Ҳзә§з”ЁжҲ·жҖҒ CPU ж—¶й—ҙпјҢд№ҹе°ұжҳҜиҝӣзЁӢзҡ„ nice еҖјиў«и°ғж•ҙдёә 1-19 д№Ӣй—ҙж—¶зҡ„ CPU ж—¶й—ҙгҖӮиҝҷйҮҢжіЁж„ҸпјҢnice еҸҜеҸ–еҖјиҢғеӣҙжҳҜ -20 еҲ° 19пјҢж•°еҖји¶ҠеӨ§пјҢдјҳе…Ҳзә§еҸҚиҖҢи¶Ҡ
  • systemпјҲйҖҡеёёзј©еҶҷдёә sysпјүпјҢд»ЈиЎЁеҶ…ж ёжҖҒ CPU ж—¶й—ҙгҖӮ
  • idleпјҲйҖҡеёёзј©еҶҷдёә idпјүпјҢд»ЈиЎЁз©әй—Іж—¶й—ҙгҖӮжіЁж„ҸпјҢе®ғдёҚеҢ…жӢ¬зӯүеҫ… I/O зҡ„ж—¶й—ҙпјҲiowaitпјүгҖӮ
  • iowaitпјҲйҖҡеёёзј©еҶҷдёә waпјүпјҢд»ЈиЎЁзӯүеҫ… I/O зҡ„ CPU ж—¶й—ҙгҖӮ
  • irqпјҲйҖҡеёёзј©еҶҷдёә hiпјүпјҢд»ЈиЎЁеӨ„зҗҶзЎ¬дёӯж–ӯзҡ„ CPU ж—¶й—ҙгҖӮ
  • softirqпјҲйҖҡеёёзј©еҶҷдёә siпјүпјҢд»ЈиЎЁеӨ„зҗҶиҪҜдёӯж–ӯзҡ„ CPU ж—¶й—ҙгҖӮ
  • stealпјҲйҖҡеёёзј©еҶҷдёә stпјүпјҢд»ЈиЎЁеҪ“зі»з»ҹиҝҗиЎҢеңЁиҷҡжӢҹжңәдёӯзҡ„ж—¶еҖҷпјҢиў«е…¶д»–иҷҡжӢҹжңәеҚ з”Ёзҡ„ CPU ж—¶й—ҙгҖӮ
  • guestпјҲйҖҡеёёзј©еҶҷдёә guestпјүпјҢд»ЈиЎЁйҖҡиҝҮиҷҡжӢҹеҢ–иҝҗиЎҢе…¶д»–ж“ҚдҪңзі»з»ҹзҡ„ж—¶й—ҙпјҢд№ҹе°ұжҳҜиҝҗиЎҢиҷҡжӢҹжңәзҡ„ CPU ж—¶й—ҙгҖӮ

  • guest_niceпјҲйҖҡеёёзј©еҶҷдёә gniceпјүпјҢд»ЈиЎЁд»ҘдҪҺдјҳе…Ҳзә§иҝҗиЎҢиҷҡжӢҹжңәзҡ„ж—¶й—ҙгҖӮ

    жҲ‘д»¬иҝҳеҸҜд»ҘйҖҡиҝҮ/proc/$pid/statзңӢжҹҗдёӘиҝӣзЁӢзҡ„cpuдҪҝз”Ёжғ…еҶөпјҢдҪҶжҳҜиҝҷдәӣйғҪжҳҜејҖжңәеҲ°зҺ°еңЁзҡ„еҺҶеҸІж—¶й—ҙгҖӮе…¶е®һжҳҜжІЎжңүеҸӮиҖғд»·еҖјзҡ„пјҢжҲ‘д»¬еә”иҜҘзңӢдёҖж®өж—¶й—ҙеҶ…зҡ„cpuдҪҝз”Ёжғ…еҶөпјҢжҲ‘д»¬дёҖиҲ¬йҮҮз”Ёд»ҘдёӢ2дёӘж–№жі•зңӢ

зі»з»ҹдјҳеҢ–-CPU-зҗҶи§ЈCPUзҡ„дёҠдёӢж–ҮеҲҮжҚў

Posted on 2020-04-19 | Edited on 2020-04-22 | In жҖ§иғҪдјҳеҢ– , CPU

д»Җд№ҲжҳҜCPUзҡ„дёҠдёӢж–ҮеҲҮжҚўпјҢзЁӢеәҸеҸҲжҳҜеҰӮдҪ•иҝҗиЎҢзҡ„пјҹ

зЁӢеәҸеңЁиҝҗиЎҢзҡ„ж—¶еҖҷпјҢCPUдјҡе°ҶдёҖйғЁеҲҶж•°жҚ®еҠ иҪҪеҲ°CPUзҡ„еҜ„еӯҳеҷЁпјҢе®ғжҳҜеұһдәҺCPUдё“еұһзҡ„й«ҳйҖҹзј“еӯҳпјҢе№¶дё”е°ҶзЁӢеәҸиҝҗиЎҢзҡ„ең°еқҖеҠ иҪҪеҲ°зЁӢеәҸи®Ўж•°еҷЁдёӯпјҢиҖҢCPUеҜ„еӯҳеҷЁ+зЁӢеәҸи®Ўж•°еҷЁз»ҹз§°дёәCPUзҡ„дёҠдёӢж–Үпјҹ
жҲ‘д»¬зҡ„зі»з»ҹдёҖиҲ¬еҸҜд»Ҙе№¶иЎҢиҝҗиЎҢеӨҡдёӘзЁӢеәҸпјҢе„ҝеӨҡдёӘиҝӣзЁӢе№¶дёҚжҳҜзңҹжӯЈзҡ„е№¶иЎҢиҝҗиЎҢзҡ„пјҢCPUдјҡеҗҢж—¶еңЁеӨҡдёӘзЁӢеәҸдёӯиҝӣиЎҢеҲҮжҚўпјҢйҖ жҲҗиҝҷз§Қе№¶иЎҢиҝҗиЎҢзҡ„вҖңй”ҷи§үвҖқпјҢиҖҢеҲҮжҚўд№ӢеүҚе°ұиҰҒзҺ°е°ҶCPUзҡ„дёҠдёӢж–Үдҝқеӯҳиө·жқҘд»ҺиҖҢдёӢж¬Ўжү§иЎҢжҳҜеҒ¶з»§з»ӯжү§иЎҢпјҢиҝҷдёӘеҸ«еҒҡCPUзҡ„дёҠдёӢж–ҮеҲҮжҚўгҖӮ

CPUдёҠдёӢж–ҮеҲҮжҚўзҡ„еҮ дёӘеңәжҷҜ

  1. иҝӣзЁӢд№Ӣй—ҙзҡ„дёҠдёӢж–ҮеҲҮжҚў
  2. зәҝзЁӢд№Ӣй—ҙзҡ„дёҠдёӢж–ҮеҲҮжҚў

  3. е“Қеә”зі»з»ҹдёӯж–ӯзҡ„дёҠдёӢж–ҮеҲҮжҚў

    жіЁпјҡ иҝҳжңүдёҖз§ҚеңәжҷҜе°ұжҳҜзі»з»ҹи°ғз”Ёзҡ„CPUдёҠдёӢж–ҮеҲҮжҚўпјҢжҲ‘д»¬зҡ„зЁӢеәҸжҳҜиҝҗиЎҢеңЁз”ЁжҲ·жҖҒзҡ„пјҢиҖҢзі»з»ҹзҡ„иө„жәҗжҳҜзі»з»ҹзҡ„еҶ…ж ёжҖҒгҖӮдёҖж¬Ўж“ҚдҪңеҫҖеҫҖйңҖиҰҒеӨҡж¬Ўзі»з»ҹи°ғз”ЁпјҢдјҡеҸ‘з”ҹеӨҡж¬ЎCPUдёҠдёӢж–ҮеҲҮжҚўжҜ”еҰӮпјҡиҜ»еҸ–дёҖдёӘж–Үд»¶жҳҜд»ҘдёӢ3жӯҘгҖӮе…Ҳopen()жү“ејҖдёҖдёӘж–Үд»¶пјҢеҶҚread()иҜ»еҸ–зЈҒзӣҳж•°жҚ®пјҢеҶҚwrite()еҶҷеҲ°ж ҮеҮҶж•°жҚ®жөҒдёӯгҖӮжҜҸдёҖж¬Ўзі»з»ҹи°ғз”Ёдјҡе…Ҳдҝқеӯҳз”ЁжҲ·жҖҒзҡ„ж•°жҚ®пјҢз„¶еҗҺеҠ иҪҪеҶ…ж ёжҖҒж•°жҚ®пјҢз„¶еҗҺеҲҮжҚўеҲ°еҶ…ж ёжҖҒпјҢж“ҚдҪңе®ҢжҲҗеҗҺпјҢеңЁеҠ иҪҪз”ЁжҲ·жҖҒж•°жҚ®пјҢеҲҮжҚўеӣһз”ЁжҲ·жҖҒз»§з»ӯд№ӢеүҚзҡ„ж“ҚдҪңгҖӮпјҲдҝ©ж¬ЎеҲҮжҚўиҝҮзЁӢпјүдёҚиҝҮжҲ‘д»¬дёҖиҲ¬з§°е‘јд»–д»¬жҳҜзү№жқғеҲҮжҚўпјҢд»–д»¬жҳҜдёҚдјҡеҸ‘з”ҹиҝӣзЁӢеҲҮжҚўзҡ„гҖӮ

    еҰӮеӣҫпјҡ

    иҝӣзЁӢй—ҙеҲҮжҚўпјҡйңҖиҰҒеҲ·ж–°зі»з»ҹзҡ„иҷҡжӢҹеҶ…еӯҳгҖҒж Ҳзӯүз”ЁжҲ·жҖҒдҝЎжҒҜпјҢеҶҚдҝқеӯҳCPUдёҠдёӢж–ҮгҖӮ
    иҝӣзЁӢеҲҮжҚўзҡ„дё»иҰҒеҺҹеӣ жңүпјҡеҲҶй…Қзҡ„иҝҗиЎҢж—¶й—ҙзүҮе·ІеҲ°пјӣзі»з»ҹиө„жәҗдёҚеӨҹгҖҗжҜ”еҰӮеҶ…еӯҳдёҚеӨҹгҖ‘зӯүеҫ…иө„жәҗж—¶еҖҷдјҡиў«жҢӮиө·пјӣи°ғз”ЁSleepеҮҪж•°пјӣжңүжӣҙй«ҳдјҳе…Ҳзә§зҡ„иҝӣзЁӢиҝӣжқҘзӯү

    зәҝзЁӢй—ҙеҲҮжҚўпјҡзәҝзЁӢжҳҜCPUи°ғеәҰзҡ„жңҖе°ҸеҚ•дҪҚпјҢиҝӣзЁӢжҳҜCPUзҡ„жңҖе°Ҹзҡ„иө„жәҗеҚ•дҪҚпјҢжүҖи°“зҡ„CPUдёҠдёӢж–ҮеҲҮжҚўе®һйҷ…жҳҜзәҝзЁӢзҡ„CPUдёҠдёӢж–ҮеҲҮжҚўеҲҶдёҖдёӢдҝ©з§Қжғ…еҶөпјҡ

    1. еҰӮжһңдҝ©дёӘзәҝзЁӢеұһдәҺеҗҢдёҖдёӘиҝӣзЁӢпјҡдёҚйңҖиҰҒеҲҮжҚўз”ЁжҲ·жҖҒдҝЎжҒҜпјҢеҶҚеҲҮжҚўCPUдёҠдёӢж–Ү
    2. еҰӮжһңдҝ©дёӘзәҝзЁӢдёҚеұһдәҺеҗҢдёҖдёӘиҝӣзЁӢпјҡйңҖиҰҒеҲҮжҚўз”ЁжҲ·жҖҒдҝЎжҒҜпјҢеҶҚеҲҮжҚўCPUдёҠдёӢж–ҮгҖӮ
      жүҖд»ҘпјҢзҺ°еңЁзЁӢеәҸеӨ§йғҪз”ЁеӨҡзәҝзЁӢеҸ–д»ЈеӨҡиҝӣзЁӢгҖӮ

    е“Қеә”зі»з»ҹдёӯж–ӯпјҡдёәдәҶе“Қеә”зЎ¬д»¶дәӢд»¶пјҢеҫҖеҫҖеӣһдёӯж–ӯеҪ“еүҚжӯЈеңЁиҝҗиЎҢзҡ„иҝӣзЁӢгҖӮе“Қеә”дёӯж–ӯеҫҖеҫҖжҜ”иҝӣзЁӢдјҳе…Ҳзә§й«ҳгҖӮ

еҰӮдҪ•е®ҡдҪҚCPUеҲҮжҚўзҡ„еңәжҷҜ

е·Ҙе…·д»Ӣз»Қ

vmstatдҪҝжҲ‘д»¬еёёз”Ёзҡ„еҲҶжһҗioзҡ„е·Ҙе…·д»–д№ҹиғҪеҲҶжһҗcpuзҡ„дёҠдёӢж–ҮеҲҮжҚўпјҢдҪҝз”Ёж–№жі•еҰӮдёӢпјҡе…¶дёӯ

  • r жҳҜrunableжҲ–иҖ…runnaingзҡ„иҝӣзЁӢж•°
  • b жҳҜblockзҡ„иҝӣзЁӢж•°
  • systemзҡ„inжҳҜдёӯж–ӯж•°
  • csжҳҜеҲҮжҚўж•°

pidstatпјҢе’ҢдёҠжңҹдёҚдёҖж ·зҡ„жҳҜеҰӮжһңжҲ‘д»¬жғізңӢcpuдёҠдёӢж–ҮеҲҮжҚўж•°йңҖиҰҒеҠ дёҠ-wпјҢ-tжҳҜжҹҘзңӢзәҝзЁӢ

  • cswch/sжҳҜжҜҸз§’иҮӘж„ҝеҲҮжҚўдёҠдёӢж–Үзҡ„ж¬Ўж•°пјҢжҜ”еҰӮпјҡж— жі•иҺ·еҸ–иө„жәҗиў«жҢӮиө·гҖӮ
  • nvcswch/sжҳҜжҜҸз§’йқһиҮӘж„ҝеҲҮжҚўдёҠдёӢж–Үзҡ„ж¬Ўж•°пјҢиҝҗиЎҢж—¶й—ҙзүҮеҲ°дәҶиў«жҢӮиө·гҖӮ

жҹҘзңӢдёӯж–ӯеҺҹеӣ watch -d cat /proc/interrupts

еҲҶжһҗй—®йўҳи·Ҝеҫ„

  1. иҮӘж„ҝдёҠдёӢж–ҮеҲҮжҚўеҸҳеӨҡдәҶпјҢиҜҙжҳҺиҝӣзЁӢйғҪеңЁзӯүеҫ…иө„жәҗпјҢжңүеҸҜиғҪеҸ‘з”ҹдәҶ I/O зӯүе…¶д»–й—®йўҳпјӣ
  2. йқһиҮӘж„ҝдёҠдёӢж–ҮеҲҮжҚўеҸҳеӨҡдәҶпјҢиҜҙжҳҺиҝӣзЁӢйғҪеңЁиў«ејәеҲ¶и°ғеәҰпјҢд№ҹе°ұжҳҜйғҪеңЁдәүжҠў CPUпјҢиҜҙжҳҺ CPU зҡ„зЎ®жҲҗдәҶз“¶йўҲпјӣ

  • дёҠзәҝеүҚе‘ҠзҹҘдёҠдёӢжёёйЈҺйҷ©

жүӢж®өпјҡдёҡеҠЎжӢҶеҲҶпјҢејӮжӯҘеҢ–

  • иҝҗиЎҢж—¶йҡ”зҰ»пјҡйҖӮз”ЁеңәжҷҜе…ұз”Ёиө„жәҗжұ зҡ„жғ…еҶөгҖӮеҰӮпјҡжңҚеҠЎе…ұз”ЁдёҖдёӘhttpclientпјҢдёҖдёӘж…ўиҜ·жұӮдјҡеҫҲеҚ з”ЁConnectionPoolзҡ„зәҝзЁӢдёҚйҮҠж”ҫ,й’ҲеҜ№жӯӨж–№жЎҲжҲ‘д»¬йңҖиҰҒжҸҗеүҚйҮҠж”ҫиө„жәҗпјҢдҝқиҜҒе…¶д»–иҜ·жұӮжӯЈеёёиҝӣиЎҢгҖӮ
