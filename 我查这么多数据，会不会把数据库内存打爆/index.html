жҲ‘жҹҘиҝҷд№ҲеӨҡж•°жҚ®пјҢдјҡдёҚдјҡжҠҠж•°жҚ®еә“еҶ…еӯҳжү“зҲҶ?

Posted on 2019-02-24 | Edited on 2019-03-01 | In Mysql , MysqlеҹәзЎҖ

е…ЁиЎЁжү«жҸҸеҜ№serverзҡ„еҪұе“Қ

дёҖжқЎиҝ”еӣһеҖјиҒҡеҲ°зҡ„sqlиҜӯеҸҘжҳҜжҖҺд№ҲеӨ„зҗҶзҡ„

еҒҮи®ҫжҲ‘д»¬жү§иЎҢдёӢйқўзҡ„иҜӯеҸҘжү§иЎҢе…ЁиЎЁжү«жҸҸпјҢиҝҷжқЎsqlиҜӯеҸҘд»ҺserverеҲ°е®ўжҲ·з«Ҝзҡ„жөҒзЁӢеҰӮдёӢ

  1. з»“жһңе…ҲеҶҷе…ҘеҲ°net_bufferдёӯпјҢз”ұеҸӮж•°net_buffer_lenghtеҶіе®ҡпјҲdef:16mпјү
  2. net_bufferеҶҷж»ЎеҗҺпјҢйҖҡиҝҮзҪ‘з»ңжҺҘеҸЈеҸ‘йҖҒеҮәеҺ»
  3. жҺҘз»ӯеҶҷе…Ҙж–°зҡ„ж•°жҚ®еҲ°net_buffer
  4. еҰӮжһңеҸ‘йҖҒеҮҪж•°йҒҮеҲ°иҝ”еӣһеҖјEAGAINзҠ¶жҖҒпјҢжҲ–иҖ…WSAEWOULDBLOCKпјҢиҜҙжҳҺжң¬ең°зҪ‘з»ңж Ҳиў«еҶҷж»ЎдәҶпјҢиҝӣе…Ҙзӯүеҫ…зҠ¶жҖҒпјҢзҹҘйҒ“зҪ‘з»ңж ҲеҸҜз”Ёз»§з»ӯеҶҷе…ҘгҖӮ

еҸҜи§ҒmysqlеҜ№дәҺжҹҘиҜўз»“жһңзҡ„еҸ‘йҖҒжҳҜиҫ№иҜ»еҸ–иҫ№еҶҷзҡ„иҝҷж ·йҳІжӯўдәҶеӨ§зҡ„sqlиҜӯеҸҘдёҖж¬ЎжҖ§иҜ»еҶҷеҜјиҮҙoom

еҰӮеӣҫжүҖзӨәпјҡ

send data to clientе’Ңsending dataзҡ„еҢәеҲ«

дёҠйқўеҸ‘йҖҒж•°жҚ®з»ҷclientиҝҮзЁӢдёӯsqlиҜӯеҸҘзҠ¶жҖҒдјҡиў«зҪ®дёәsend data to clientзҠ¶жҖҒпјҢиҰҒе’Ңsending dataеҢәеҲҶејҖ

sending dataжҳҜеҪ“дёҖжқЎsqlиҜӯеҸҘжү§иЎҢж—¶еҖҷе°ұиў«и®ҫзҪ®дёәиҜҘзҠ¶жҖҒпјҢзҹҘйҒ“иҝҷжқЎиҜӯеҸҘз»“жқҹпјҢжүҖд»Ҙsending dataе№¶дёҚиғҪиЎЁзӨәиҝҷжқЎиҜӯеҸҘжӯЈеңЁеҫҖе®ўжҲ·з«ҜеҸ‘йҖҒж•°жҚ®

еҪ“ж•°жҚ®з»§з»ӯејҖе§ӢеҫҖе®ўжҲ·з«ҜеҸ‘йҖҒж—¶еҖҷж•°жҚ®еӨ„дәҺsend data to clientйҳ¶ж®өгҖӮ
еҪ“жү§иЎҢеҷЁејҖе§Ӣжү§иЎҢsqlиҜӯеҸҘж—¶еҖҷпјҢиҝҷжқЎsqlиҜӯеҸҘеӨ„дәҺsending dataйҳ¶ж®ө

mysql_store_result

дёҖиҲ¬жҲ‘д»¬йғҪдјҡйҮҮз”ЁеҸӮж•°пјҡmysql_store_resultзӣҙжҺҘжҠҠжҹҘиҜўз»“жһңдҝқеӯҳеҲ°жң¬ең°пјҢиҖҢдёҚжҳҜmysql_use_result,иҜ»дёҖиЎҢеӨ„зҗҶдёҖиЎҢпјҢеӣ дёәеҰӮжһңдёҖдёӘдәӢеҠЎйҖ»иҫ‘еҫҲеӨҚжқӮеҫҲеҸҜиғҪиҰҒеӨ„зҗҶеҫҲд№…жүҚдјҡеҺ»иҜ»дёӢдёҖиЎҢ

е…ЁиЎЁжү«жҸҸеҜ№innodbзҡ„еҪұе“Қ

иҝҳи®°еҫ—Mysqlжӣҙж–°йҮҮз”ЁдәҶBUFF_POOL+WALзҡ„йЎәеәҸеҶҷжңәеҲ¶жӣҝд»ЈдәҶпјҢйҡҸжңәеҶҷпјҢеҠ йҖҹдәҶеҶҷзҡ„жҖ§иғҪд№Ҳпјҹе…¶е®һBUFF_POOLзҡ„иҝҷдёӘжңәеҲ¶иҝҳеҠ йҖҹдәҶжҹҘиҜўзҡ„жҖ§иғҪгҖӮжҜ”еҰӮдёҖдёӘsqlиҜӯеҸҘжҹҘиҜўж•°жҚ®зҡ„ж—¶еҖҷпјҢеҰӮжһңж•°жҚ®йЎөеңЁеҶ…еӯҳдёӯпјҢйӮЈд№Ҳе°ұдёҚз”Ёжү«жҸҸзЈҒзӣҳдәҶпјҢдёҖиҲ¬дёҖдёӘзЁіе®ҡзҡ„зәҝдёҠдёҡеҠЎbuff_poolзҡ„зЈҒзӣҳе‘ҪдёӯзҺҮйғҪдјҡеҫҲй«ҳжҲ‘д»¬еҸҜд»Ҙз”Ёе‘Ҫд»Өshow engine innodb statusжқҘжҹҘзңӢе‘ҪдёӯзҺҮгҖӮ

йӮЈд№ҲеҰӮжһңе…ЁиЎЁжү«жҸҸдёҖдёӘеҫҲеӨ§зҡ„иЎЁпјҢи§ҰеҸ‘дәҶbuff_poolзҡ„Lruз®—жі•жҳҜеҗҰеҫҲжһҒеӨ§зҡ„йҷҚдҪҺbuff_poolзҡ„е‘ҪдёӯзҺҮпјҢеўһеӨ§зЈҒзӣҳзҡ„ејҖй”Җе‘ўпјҹmysqlжҳҜйҮҮз”Ёд»Җд№Ҳзӯ–з•ҘдјҳеҢ–иҝҷз§Қжғ…еҶөзҡ„е‘ўпјҹ

Mysqlзҡ„lruз®—жі•еҰӮеӣҫпјҡ

  1. mysqlеҗ§buff_poolеҲҶдәҶyoungе’Ңoldд»ЈпјҢеҚ жҜ”жҳҜ5:3
  2. еҰӮжһңе‘Ҫдёӯзҡ„ж•°жҚ®йЎөеңЁyoungд»Јдјҡиў«з§»еҠЁеҲ°й“ҫиЎЁзҡ„headеӨ„
  3. еҰӮжһңж–°жҸ’е…ҘдәҶдёҖдёӘж•°жҚ®йЎөпјҢPx,дјҡж·ҳжұ°жңҖйқ иҝ‘tailзҡ„pm,иҝҷж—¶еҖҷpxдјҡжҸ’е…ҘеҲ°oldд»Јзҡ„еӨҙйғЁпјӣ
  4. еӨ„дәҺoldеҢәеҹҹзҡ„ж•°жҚ®йЎөиў«и®ҝй—®ж—¶еҖҷпјҢMysqlдјҡжңүдёҖдёӘзӯ–з•Ҙ
    1. еҸ‘зҺ°ж•°жҚ®йЎөе·Із»ҸеңЁoldеҢәеӯҳеңЁдәҶ1з§’д»ҘдёҠдәҶпјҢдјҡз§»еҠЁеҲ°young
    2. еҸҚд№ӢпјҢдёҚз§»еҠЁпјҢж…ўж…ўиў«ж·ҳжұ°

mysqlз”Ёиҝҷз§Қзӯ–з•ҘжқҘдјҳеҢ–е…ЁиЎЁжү«жҸҸпјҢеӣ дёәе…ЁиЎЁжү«жҸҸж—¶еҖҷиҝҷдәӣж•°жҚ®йЎөдјҡйЎәеәҸиҜ»пјҢжүҖд»ҘиӮҜе®ҡдёҚдјҡи¶…иҝҮ1з§’пјҢжүҖд»ҘдёҚдјҡз§»еҠЁеҲ°youngпјҢеҸӘдјҡеңЁoldд»Јж…ўж…ўиў«ж·ҳжұ°пјҢиҖҢдё”ж·ҳжұ°еҸӘдјҡйҷҗеҲ¶еңЁoldд»ЈпјҢеҜ№youngе®Ңе…Ёж— еҪұе“ҚпјҲжӯЈеёёдёҡеҠЎзҡ„жҹҘиҜўпјү
